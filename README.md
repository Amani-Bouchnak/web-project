# Web Project – Average Calculator

A simple web application that calculates students’ averages and provides personalized academic advice.

Une application web simple qui calcule la moyenne des élèves et fournit des conseils académiques personnalisés.

## Features
- Calculate student averages by section
- Display subject-specific inputs
- Provide a clean and simple interface

## Built with
- HTML
- CSS
- JavaScript

## Note
This project was developed as a learning exercise with the assistance of AI tools.
